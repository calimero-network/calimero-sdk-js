Decorators

@State

Marks a class as the application state container. Only one @State class per service.

@State
export class MyApp {
  items: UnorderedMap<string, string> = createUnorderedMap();
}

@Logic

Links a logic class to its state class. Methods in this class become callable service endpoints.

@Logic(MyApp)
export class MyAppLogic extends MyApp {
  // methods become RPC endpoints
}

@Init

Marks a static method as the service initialiser. Called once when the service is first installed.

@Init
static initialize(): MyApp {
  return new MyApp();
}

@View()

Marks a method as read-only. Skips persistence pipeline — no state delta emitted, no storage write.

@View()
getItems(): string[] {
  return this.items.keys();
}

@Event

Marks a class as an event type. Event fields are serialised with ABI-aware Borsh for Rust compatibility.

@Event
export class ItemAdded {
  constructor(
    public key: string,
    public value: string
  ) {}
}

Environment Functions

Import from @calimero-network/calimero-sdk-js/env:

Identity & Time

log(message: string)
Logs a message to the runtime.
contextId(): Uint8Array
Current context ID (32 bytes).
executorId(): Uint8Array
Current executor identity key (32 bytes).
timeNow(): bigint
Current timestamp in nanoseconds.
import * as env from '@calimero-network/calimero-sdk-js/env';

env.log('hello');
const ctx = env.contextId();
const exec = env.executorId();
const now = env.timeNow();

Context Management

contextAddMember(publicKey)
Add member by 32-byte Ed25519 public key. Takes effect after successful execution.
contextRemoveMember(publicKey)
Remove member. Takes effect after successful execution.
contextIsMember(publicKey): boolean
Synchronous read — checks membership.
contextMembers(): Uint8Array[]
All member public keys in the current context.
contextCreate(protocol, appId, initArgs, alias)
Creates a child context with the given protocol, app ID, init args, and alias (max 64 bytes).
contextDelete(contextId)
Delete a context. Pass current context ID for self-deletion.
contextResolveAlias(alias): Uint8Array | null
Resolve a context alias to its 32-byte context ID.
import {
  contextAddMember, contextIsMember,
  contextMembers, contextCreate
} from '@calimero-network/calimero-sdk-js/env';

const key = new Uint8Array(32);
contextAddMember(key);
const isMember = contextIsMember(key);
const members = contextMembers();

// Create child context
contextCreate(
  new TextEncoder().encode('near'),
  new Uint8Array(32), // app ID
  new TextEncoder().encode('{}'),
  new TextEncoder().encode('my-ctx')
);

Storage

storageRead(key: Uint8Array): Uint8Array | null
Read raw bytes from node-local storage.
storageWrite(key: Uint8Array, value: Uint8Array): void
Write raw bytes to node-local storage.
storageRemove(key: Uint8Array): boolean
Remove a key from storage. Returns true if it existed.
const key = new TextEncoder().encode('my-key');
const val = new TextEncoder().encode('data');

env.storageWrite(key, val);
const read = env.storageRead(key);
env.storageRemove(key);

Cryptography

ed25519Verify(sig, pubKey, msg): boolean
Verify a 64-byte Ed25519 signature against a 32-byte public key and message bytes.
const sig = new Uint8Array(64);
const pub = new Uint8Array(32);
const msg = new TextEncoder().encode('hello');

const valid = env.ed25519Verify(sig, pub, msg);
if (!valid) throw new Error('bad signature');

Events

emit(event: AppEvent): void
Emit an event without a handler.
emitWithHandler(event, handler: string): void
Emit with a named handler method on receiving nodes.
import { emit, emitWithHandler } from '@calimero-network/calimero-sdk-js';

emit(new ItemAdded('key', 'value'));
emitWithHandler(new ItemAdded('key', 'val'), 'onItemAdded');

State Helper Factories

Import from @calimero-network/calimero-sdk-js. Always initialise CRDT fields inline using these factories so the runtime rehydrates persisted collection IDs:

import {
  createUnorderedMap, createUnorderedSet,
  createVector, createCounter,
  createLwwRegister, createUserStorage,
  createFrozenStorage, createPrivateEntry
} from '@calimero-network/calimero-sdk-js';

// Standard CRDT collections
const map    = createUnorderedMap<string, number>();
const set    = createUnorderedSet<string>();
const vec    = createVector<string>();
const ctr    = createCounter();
const reg    = createLwwRegister<string>();

// Specialised storage
const users = createUserStorage<UserProfile>();
const frozen = createFrozenStorage<Document>();

// Node-local (no CRDT replication)
const priv = createPrivateEntry<{ token: string }>('private:key');
